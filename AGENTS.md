# eos-engine

Browser-based trading card game engine (Yu-Gi-Oh! Forbidden Memories style).

## Quick Reference

- **Package Manager:** npm
- **Dev:** `npm run dev`
- **Test:** `npm run test`
- **Build:** `npm run build`
- **Typecheck:** `npx tsc --noEmit`
